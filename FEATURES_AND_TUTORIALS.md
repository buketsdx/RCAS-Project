## Company Management

### What is Company Management?
RCAS allows you to manage multiple companies in a single application. Each company has completely separate data - no mixing or cross-contamination. Perfect for managing multiple business units, franchises, or accounting firms with multiple clients.

### Creating a New Company
1. Go to **Company Management** from the sidebar
2. Click the **Add** button (top-right, green)
3. Fill in the company details:
   - Company Name (required)
   - Company Type (Sole Proprietor, Partnership, LLP, Private Limited, etc.)
   - GST Number (optional but recommended)
   - Contact Number
   - Email Address
   - Address
   - City, State, Postal Code
   - Country
   - Bank Account (optional)
   - Industry Type
   - Financial Year Start (MM/DD)
   - Financial Year End (MM/DD)
   - Password (optional - for security)
4. Click **Create** button
5. Company is now available for use

### Switching Between Companies
**Method 1: Using Top Navigation**
1. Look at the top navigation bar
2. Click on your company name or the dropdown arrow
3. Select the company you want to switch to
4. If it has a password, enter it
5. Data automatically filters to show only that company's information

**Method 2: Using Company Management**
1. Go to **Company Management** from sidebar
2. Find the company in the list
3. Click the **Switch** button (bottom-right of the row)
4. Enter password if required
5. You're now switched to that company

**Method 3: Using Keyboard Shortcut**
- Press **Ctrl + Alt + C** to open company switcher
- Use arrow keys to select company
- Press **Enter** to switch
- Type password if needed and press **Enter**

### Setting Up Company Password Protection
1. Go to **Company Management**
2. Click **Edit** button (pencil icon) on the company
3. Scroll down to the **Password** field
4. Enter a strong password (e.g., CompanyName@2024)
5. Click **Update** button
6. From now on, switching to this company requires the password
7. To remove password: Edit company, clear the password field, click Update

### Understanding Company-Specific Data
When you switch companies:
- **Branches** list shows only branches for that company
- **Ledgers** list shows only ledgers for that company
- **Stock Items** list shows only items for that company
- **Invoices/Orders** show only transactions for that company
- **Reports** automatically filter by company
- All new records created belong to the selected company

âœ… Use descriptive company names for easy identification
âœ… Always verify you're in the correct company (check top bar) before creating records
âœ… Document company passwords securely
âœ… Export company data regularly as backup

## Logo Upload & Image Editor

### Uploading a Company Logo
1. Go to **Company Info** from the sidebar
2. Select your company from the dropdown (if multiple companies)
3. Scroll to the **Logo** section
4. Click **Upload Logo** button
5. Choose an image file from your computer (JPG, PNG, GIF, WebP)
6. The image automatically opens in the Image Editor

### Using the Image Editor

#### Basic Adjustments
The Image Editor provides several adjustment tools:

**Brightness Slider**
- Drag left to darken the image
- Drag right to brighten the image
- Range: -100 to +100

**Contrast Slider**
- Drag left to decrease contrast (flatter colors)
- Drag right to increase contrast (more vivid)
- Range: -100 to +100

**Saturation Slider**
- Drag left to desaturate (more grayscale)
- Drag right to saturate (more colorful)
- Range: -100 to +100

**Rotation Slider**
- Drag left for counter-clockwise rotation
- Drag right for clockwise rotation
- Range: 0Â° to 360Â°

**Scale Slider**
- Drag left to zoom out (smaller view)
- Drag right to zoom in (larger view)
- Useful for focusing on specific parts

#### Using the Crop Tool
1. Click **Enable Crop** button to activate crop mode
2. On the preview image:
   - **Drag** to draw a crop box (drag from top-left to bottom-right)
   - **Move**: Click inside the crop box and drag to reposition
   - **Resize**: Drag corner handles to resize the crop area
3. Once satisfied, click **Apply Crop**
4. The image is cropped to your selection

#### Saving Your Logo
1. After making all adjustments, click **Save Logo** button
2. The logo is saved and associated with your company
3. The logo will appear on:
   - Invoices and receipts
   - Reports header
   - Company info pages
   - Letterheads (when printing)

## Basic Operations

### Adding New Records (Generic Instructions)
1. Navigate to the page where you want to add data
2. Click **Add** button (usually green, top-right)
3. Fill in all required fields (marked with *)
4. Optional fields can be left empty
5. Click **Create** to save
6. See success message
7. Record appears in list

### Editing Existing Records
1. Find the record in the list
2. Click **Edit** button (pencil icon)
3. Form opens with current values
4. Modify fields as needed
5. Click **Update** to save changes
6. List refreshes automatically

### Deleting Records
1. Find the record in the list
2. Click **Delete** button (trash icon)
3. Confirmation dialog appears
4. Click **Delete** to confirm
5. Record is permanently removed
6. Success message confirms

### Searching Records
1. Each list has a **Search** box (magnifying glass icon)
2. Start typing to search
3. Search works across all columns
4. Results update instantly
5. Clear search to see all records again
6. Use pagination to navigate results

## Reports & Analytics

### Accessing Reports
1. Go to **Accounts & Reports** section
2. Select the report type you want to view
3. Choose date range if applicable
4. Apply any filters if needed
5. View report on screen with charts

### Common Reports
- **Balance Sheet**: Assets, liabilities, equity at specific date
- **Profit & Loss**: Income and expenses for period
- **Trial Balance**: All ledger accounts and balances
- **Daybook**: Daily transaction summary
- **Ledger Report**: Detailed account transactions

### Exporting Reports
1. Open the report
2. Click **Download** button
3. Choose format: PDF or Excel
4. File downloads to your computer
5. Open with Excel, Google Sheets, or PDF viewer

### Printing Reports
1. Open the report
2. Click **Print** button
3. Browser print dialog opens
4. Customize printer, page size, orientation
5. Click Print button in dialog
6. Report prints to selected printer
